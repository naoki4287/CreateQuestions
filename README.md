# 問題クリエイター

## 概要

問題クリエイターは問題を作成し、解答できるアプリです。

## 機能一覧

-   問題作成機能
-   問題編集機能
-   問題削除機能
-   解答機能
-   出題数や出題形式をカスタマイズできる設定機能
-   認証機能
-   リトライ機能
-   正誤判定機能
-   回答結果表示機能

## 使用した技術

フロントエンド
-   HTML/CSS
-   JavaScript(バニラ)
-   Tailwind CSS 3.0.18

バックエンド
-   PHP 8.0.16
-   Laravel 9.7.0

インフラ
-   AWS (VPC, EC2, Route53, RDS, IAM, Amazon Linux 2)
-   Apache 2.4.53
-   MySQL 8.0.28

使用したツール
-   GitHub
-   VSCode
